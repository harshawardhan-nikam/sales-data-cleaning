# 🧹 Sales Data Cleaning Project

## 📌 Project Overview
This project performs data cleaning and preprocessing on a Sales Dataset obtained from Kaggle.

The objective is to prepare raw sales data by handling duplicates, formatting inconsistencies, and correcting data types.

---

## 📂 Dataset
- Name: Sales Data
- Source: Kaggle
- Rows: 1000
- Columns: 14

---

## 🛠 Tools Used
- Python
- Pandas

---

## ✅ Cleaning Steps Performed

1. Removed duplicate records.
2. Standardized text columns (lowercase, removed extra spaces).
3. Converted `Sale_Date` column to datetime format.
4. Renamed all column headers to lowercase.
5. Verified correct data types.
6. Exported cleaned dataset.
